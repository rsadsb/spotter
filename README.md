# spotter
```
Rust ADS-B processor and web server providing information in json format

Usage: spotter --serve-addr <SERVE_ADDR> --dump1090-addr <DUMP1090_ADDR> <LAT> <LONG>

Arguments:
  <LAT>
  <LONG>

Options:
  -s, --serve-addr <SERVE_ADDR>
  -d, --dump1090-addr <DUMP1090_ADDR>
  -h, --help
```
